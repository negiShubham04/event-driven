# Event Driven Architecture - Java (ASynchronous In-Memory Implementation)

This project demonstrates the **fundamentals of Event Driven Architecture (EDA)** using a **pure Java in-memory asynchronous event bus**.

The goal of this implementation is to build conceptual clarity around how services communicate using events before introducing asynchronous execution and distributed messaging systems like Kafka or RabbitMQ.
